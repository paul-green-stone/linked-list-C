Detailed Description

The methods struct encapsulates all the methods available for a list. It serves as a container for the function pointers that define the behavior and operations supported by the list, providing a unified interface to interact with the list's functionality.

The methods struct is defined as an incomplete data type, which means that its function pointers are not specified within the struct definition. Instead, the function pointers are defined separately in the code where the methods struct is used.

void(* methods::destroy) (void *data)

Provides a way to free dynamically allocated data when sList_destroy is called.

Provides a way to free dynamically allocated data when sList_destroy is called. For example, if the list contains data dynamically allocated using malloc, destroy should be set to free to free the data as the linked list is destroyed. For structured data containing several dynamically allocated members, destroy should be set to a user-defined function that calls free for each dynamically allocated member as well as for the structure itself. For a linked list containing data that should not be freed, destroy should be set to NULL.

void(* methods::print) (void *data)

Provides a way to display Node's data.

The print method is used to output the data held by each node in the singly-linked list. It does this by traversing the list and calling the user-defined print function for each node's data.
